# Парсер вакансий с Habr Career

Скрипт автоматически собирает вакансии с сайта Habr Career по ключевому слову "Python" и сохраняет их в Excel-файл с детальной информацией.

## Используемые технологии
- **Python 3**
- **Requests** — для отправки HTTP-запросов
- **BeautifulSoup4** — для парсинга HTML
- **Pandas** — для обработки и сохранения данных в Excel

##  Как запустить
1. Убедитесь, что установлены необходимые библиотеки:pip install requests beautifulsoup4 pandas openpyxl
2. Запустите скрипт:python parser_habr.py
3. В папке проекта появится файл `vacancies_habr_YYYYMMDD_HHMM.xlsx` с собранными вакансиями.

## Что делает скрипт
- Название вакансии
- Компания
- Зарплата
- Ссылка на вакансию
